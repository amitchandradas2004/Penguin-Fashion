# 🐧 Penguin Fashion – eCommerce Landing Page

A modern and responsive **fashion eCommerce landing page** designed to showcase winter jackets for men and women. This project focuses on clean UI design, responsive layouts, and attractive product cards.

## 📌 Overview

Penguin Fashion is a simple front-end project that displays a clothing store interface with product sections, promotional content, and a clean shopping experience layout.
The design highlights fashion products while maintaining a minimal and modern user interface.

## 🌐 [Live Preview](https://amitchandradas2004.github.io/Penguin-Fashion/)

## ✨ Features

- Responsive design

- Product cards with pricing
